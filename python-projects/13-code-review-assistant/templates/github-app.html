📚 How to Set Up a GitHub App

Follow these steps to create and configure a GitHub App for automatic PR analysis:

  • Go to GitHub Settings → Developer settings → GitHub Apps → New GitHub App
  • Set a unique name (e.g., "My Code Review Assistant")
  • Set Homepage URL to your server URL (e.g., http://localhost:8000)
  • Set Webhook URL to: http://localhost:8000/api/github/webhook
  • Generate a webhook secret and save it to your .env file as GITHUB_WEBHOOK_SECRET
  • Set Repository permissions:
    • Contents: Read-only (to fetch PR diffs)
    • Pull requests: Read & write (to analyze PRs and post comments)
  • Subscribe to events: Pull request, Push
  • Create the app and generate a private key (download the PEM file)
  • Note your App ID and paste it along with the private key below
