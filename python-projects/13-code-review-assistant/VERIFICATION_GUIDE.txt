===========================================
DATABASE VERIFICATION
===========================================

Run this to verify the database has issues:

```bash
cd /path/to/project
python -c "
from src.core.database import DatabaseManager, Repository, CodeFile, Issue

db_manager = DatabaseManager()
with db_manager.get_session() as db:
    repo = db.query(Repository).filter(
        Repository.name == 'adaptive-suggestion-engine'
    ).first()

    issues = db.query(Issue).join(CodeFile).filter(
        CodeFile.repository_id == repo.id
    ).all()

    print(f'Repository: {repo.name}')
    print(f'Total issues: {len(issues)}')

    # Show breakdown
    from collections import Counter
    severities = Counter(i.severity.value for i in issues)
    categories = Counter(i.category.value for i in issues)

    print('\nSeverity breakdown:')
    for sev, count in severities.items():
        print(f'  {sev}: {count}')

    print('\nCategory breakdown:')
    for cat, count in categories.items():
        print(f'  {cat}: {count}')
"
```
